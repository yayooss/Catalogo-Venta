### CATALOGO-VENTA
<p align="justify">
  This project is a web application built with **NiceGUI and PocketBase**, designed to display and manage a product catalog for sale.
  The products come from company inventory that is no longer in use and are showcased to potential buyers through a modern interface with a shopping cart and admin panel.
</p>

### TECHNOLOGIES USED
- Python.
    - NiceGUI.
- PocketBase.
- JSON.

### FEATURES
#### CLIENT SIDE
- Product catalog browsing.
- Search by product name.
- Detailed product view (images, description, characteristics).
- Shopping cart with quantity management.
- Interest request submission (email & phone).
